# WIFISDIO
Port of nocash's [wifiboot](https://problemkaputt.de/wifiboot.htm) wifisdio driver to C, in order to provide a normal API and to learn how it works.

##### Done
- SDIO Initialization
- SDIO Channel scanning + BSS Connect
- ARP + IPv4 + ICMP
